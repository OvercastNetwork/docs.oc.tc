The slot attribute of items accepts both the slot number and Mojang slot names, e.g. slot.hotbar.8, armor.head, 35, and hotbar.8 will all work.

slot.weapon.mainhand is an automatic slot, that means that when inserting an item using this slot it will be handled as if no slot is specified.

Mojang Slot Name Type
slot.armor.chest Armor
slot.armor.feet Armor
slot.armor.head Armor
slot.armor.legs Armor
slot.weapon.mainhand Hand
Auto
slot.weapon.offhand Hand
slot.hotbar.[0 to 8] Inventory
slot.inventory.[0 to 26] Inventory

Material Matchers

Depending on the material/item element used a material or multiple materials can be specified with or without data values.
Material Values
Value Type Description
Single Material Pattern
 A single material pattern consisting of a material_name:data_value string.
Material Pattern
 A pattern that supports multiple materials with a optional data value.
Multiple materials can be listed by separating them with a semicolon;.
material_name:data_value;material_name:data_value
Material Name
 A single material.
Does not accept a damage/data value.
